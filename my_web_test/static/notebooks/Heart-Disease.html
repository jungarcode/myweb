EDA Heart-Disease¶

This dataset contains medical information from 302 patients who had heart failure or who are at risk of heart attacks. This data was collected during a follow-up period, where each patient profile has 13 clinical characteristics.

Using the information in the dataset, an exploratory data analysis (EDA) will be performed, which will allow us to visualize which characteristics most affect the patient with this pathology and then perform a classification with the Logistic Regression model that allows us to calculate the probability of the patient dying during a period of time. The Target variable is coded as 1 if the patient is sick and 0 if he is not.

Feature Details

https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset

  • Age (age in years)
  • Sex (1 = male; 0 = female)
  • CP (chest pain type)
    • Value 1: typical angina
    • Value 2: atypical angina
    • Value 3: non-anginal pain
    • Value 4: asymptomatic
  • TRESTBPS (resting blood pressure (in mm Hg on admission to the hospital))
  • CHOL (serum cholestoral in mg/dl)
  • FPS (fasting blood sugar > 120 mg/dl) (1 = true; 0 = false)
  • RESTECH (resting electrocardiographic results)
  • THALACH (maximum heart rate achieved)
  • EXANG (exercise induced angina (1 = yes; 0 = no))
  • OLDPEAK (ST depression induced by exercise relative to rest)
  • SLOPE (the slope of the peak exercise ST segment)
  • CA (number of major vessels (0-3) colored by flourosopy)
  • THAL (3 = normal; 6 = fixed defect; 7 = reversable defect)
  • TARGET (1 or 0)
